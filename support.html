FAQ

  • Jak se připojit? add
    V sekci multiplayer vyber direct connect a vlož tam tuto IP adresu `play.midway.mc`
  • Je server i pro neoriga? add
    Ano, na server se lze připojit i s neoriginální kopií minecraftu.
  • Mohu server podpořit? add
    Ano, na našem discordu si můžeš zakoupit různé věci do hry a tím nás podpoořit.
  • Dostal jsem ban, co mám dělat? add
    Pokud si myslíš, že jsi dostal ban neprávem, založ si ticket na našem discordu a popiš svou situaci
